# About
Omcoin is a token that encourages everyone to embark on a journey of personal growth through its meditation reward system.

Omcoin is a token of exchange in an opt-in advertising system, entailing
sponsored meditations based within its mobile app and Web counterparts.

Omcoin can be used for less expensive customer listening attention and pronounced recognition for creatives, purchasing premium content within Omapp, digital goods such as gift cards and other utilities in our self-development marketplace.
